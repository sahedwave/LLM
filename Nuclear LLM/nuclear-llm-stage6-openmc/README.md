# nuclear-llm-stage6-openmc

Stage 6 tool-calling and OpenMC-style verification layer for a nuclear LLM.

## Included

- rule-based intent parsing
- safe template-backed config generation
- sandbox validation
- deterministic proxy simulation backend
- simulation alignment scoring
- feedback refinement
- Stage 5 integration bridge

## Quick start

```bash
python -m unittest discover -s tests
```
